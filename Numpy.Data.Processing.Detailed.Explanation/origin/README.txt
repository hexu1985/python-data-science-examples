●Python的安装
　如果是macOS，请参照本书的Pvi。
　如果是Windows的话，可以在官方网站的以下URL下载Python Windows版文件的一览表齐全。

・Python Releases for Windows
URL 　https://www.python.org/downloads/windows/

　从中寻找自己想要安装的版本，Windows x86-64的标记后，下载显示为executable installer的请做。
　版本是3.5以后的，本书的代码可以正常运行。双击下载的文件运行的话安装程序就会启动，所以请按照画面的指示进行Python的安装。

●IPython和库的安装
　本书的P.003介绍了iPython安装方法，P.007介绍了库的安装方法。
这里也要提一下。
    输入pip命令可以安装IPython或NumPy。

［终端窗口］
-------------------------------
$ pip install ipython
-------------------------------

［终端窗口］
-------------------------------
$ pip install numpy
-------------------------------

     指定并安装库的版本时，

-------------------------------
pip install <库名>==<版本名>
-------------------------------

请输入。
如果你是Numpy 1.14.3版本，请按以下步骤输入。

［终端窗口］
-------------------------------
$ pip install numpy==1.14.3
-------------------------------
